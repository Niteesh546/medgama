# 🩺 MedGamma — Medical Image Analysis

AI-powered medical image analysis for disease detection and treatment suggestions, powered by **Google Gemini 2.0 Flash**.

## Features
- 🔬 **Disease Detection** — Identifies conditions from medical images
- 💊 **Treatment Suggestions** — Recommends treatments and next steps
- 🏥 **Multi-modal Support** — Chest X-rays, dermatology, ophthalmology, pathology, MRI, CT
- ⚡ **Fast** — Results in 2-5 seconds
- 🆓 **Free** — Uses Gemini API free tier (1500 requests/day)

## How to Use
1. Get a free Gemini API key at [Google AI Studio](https://aistudio.google.com/apikey)
2. Upload a medical image (X-ray, skin condition, fundus, etc.)
3. Paste your API key
4. Click **Analyze Image**
5. Review the AI-powered analysis

## Disclaimer
This tool is for **informational purposes only**. Always consult a qualified healthcare professional for diagnosis and treatment. Do not rely on this tool for clinical decisions.

- **AI Model**: Google Gemini 2.0 Flash
